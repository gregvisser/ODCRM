# ODCRM Training Docs

## Start here
- New operator: read [ODCRM_USER_WORKFLOW_OVERVIEW.md](./ODCRM_USER_WORKFLOW_OVERVIEW.md)
- Full manual index: read [ODCRM_USER_TRAINING_MASTER_INDEX.md](./ODCRM_USER_TRAINING_MASTER_INDEX.md)
- Exact task steps: use [ODCRM_TASK_PLAYBOOKS.md](./ODCRM_TASK_PLAYBOOKS.md)
- When the UI feels confusing: check [ODCRM_KNOWN_LIMITS_AND_GOTCHAS.md](./ODCRM_KNOWN_LIMITS_AND_GOTCHAS.md)

## How to use this folder
1. Read the workflow overview first.
2. Read the relevant module guide for the tab you are in.
3. Use the playbooks while doing the task in ODCRM.
4. Use the gotchas file when expected results do not match what you see.
