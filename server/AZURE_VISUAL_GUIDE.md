# Azure Portal - Visual Guide to Find App Registrations

## üéØ Quick Method (Most Reliable)

### Step 1: Open Azure Portal
- Go to: https://portal.azure.com
- Sign in with your Microsoft account

### Step 2: Use the Top Search Bar
- Look at the **very top** of the Azure Portal page
- You'll see a search bar that says: "Search resources, services, and docs"
- **Type exactly:** `App registrations`
- Press Enter or click the search result

### Step 3: You Should See
- A page titled **"App registrations"**
- A blue button that says **"+ New registration"** (usually top left)
- A list/grid of existing app registrations (may be empty if first time)

## ‚ùì Common Issues

### "I don't see the search bar"
- Make sure you're signed in
- Try refreshing the page (F5)
- The search bar is always at the top of Azure Portal

### "Search doesn't return results"
- Try typing just: `entra`
- Or try: `aad`
- Make sure you have an active Azure subscription

### "I see 'Microsoft Entra ID' but no 'App registrations'"
- Click on **Microsoft Entra ID**
- Look in the **left sidebar menu**
- Scroll down - it's usually in the middle of the menu
- Should be under "Manage" section

### "I get an error about permissions"
- Make sure you're using an account with Azure access
- You may need to be added to an Azure AD tenant
- Free Azure account works fine
